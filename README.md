# Frontend - TEA Gestión

Frontend desarrollado con Next.js, TypeScript y Tailwind CSS para la plataforma de gestión interna.

## Requisitos

- Node.js
- npm
- Backend arrancado en `http://localhost:8080`
- Base de datos MariaDB disponible mediante Docker o entorno local

## Configuración

Crear un archivo `.env.local` a partir de `.env.local.example`:


NEXT_PUBLIC_API_URL=http://localhost:8080/api/v1

## Instalación

- npm install

## Desarrollo

- npm run dev

La aplicación estará disponible en:

 http://localhost:3000

## Comprobación de calidad

- npm run lint

## Build de producción

- npm run build
